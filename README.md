# Lavínia A. N. de Queiroz — Portfolio

Site pessoal de portfólio. Minimalista, clean e preparado para crescer.

**Stack:** HTML · CSS · JavaScript puro (sem dependências)

## Como rodar localmente

```bash
git clone https://github.com/lavsqueiroz/portfolio.git
open index.html
```

## Deploy com GitHub Pages

1. Vá em **Settings → Pages**
2. Source: **Deploy from branch → main**
3. O site estará em: `https://lavsqueiroz.github.io/portfolio`
